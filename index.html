🌐 Publicar el diagnóstico en línea

Para tener un enlace propio que puedas compartir con cada emprendedor, tienes dos opciones gratuitas:

⚡ Opción 1: Netlify Drop (2 minutos)

  1. Ve a app.netlify.com/drop
  2. Arrastra el archivo icone-dx-madurez-digital.html a la zona gris
  3. ¡Listo! En segundos tienes una URL pública

🐙 Opción 2: GitHub Pages (URL permanente)

  1. Sube el archivo como index.html a un repositorio
  2. Ve a Settings → Pages → Source: "main" → Save
  3. URL: tu-usuario.github.io/icone-dx
